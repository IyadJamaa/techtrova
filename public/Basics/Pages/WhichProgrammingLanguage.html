Which Programming Language To Learn

A guide on how to pick the right programming language for you.


Introduction

There are many programming languages to learn and over the course of your career you will find yourself switching between many of them, for now lets stick to one language, you have 2 options :

option one (which is what we recommend):
research and pick a track you're interested in and learn the programming language required for this track, while learning the rest of the basics on our website, there are also many tracks available for you to pick and we made a specific section for each track where you can learn more about each individual track to help you. the available tracks are :

  • Web Development
  • App Development


Option two :
start with learning C++ as it will improve your foundation in programming and problem solving.

Problem Solving

after watching the courses for the programming language you chose and in order to have a solid understanding of any programming language you need to practice problem solving consistently, try to make it a daily habit.


we recommend by starting to solve very easy - easy problems on Edabit with the programming language you just learned.
